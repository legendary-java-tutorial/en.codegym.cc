For loop in Pascal

----------------------------------------

I must tell you. A programming language that has no for loop doesn't have much. It is so very useful. And Pascal—the favorite language of high school students, college students, and the entire educational system on Planet Earth in the 20th century—was no exception. We present for your consideration verbose and precise Pascal code for loops and their literal translation into our favorite Java.

----------------------------------------

"Amigo, you won't believe it, but there is a For loop in the Pascal language too. This construct exists in almost every programming language. But in Pascal it's much clearer. Look:"

Pascal
For i := 1 to 10 do
Begin
    Writeln(i);
End;
Java
for (int i = 1; i <= 10; i++)
{
    System.out.println(i);
}
Pascal
For i := 1 to 10 do step 2
Begin
    Writeln(i);
End;
Java
for (int i = 1; i <= 10; i = i + 2)
{
    System.out.println(i);
}
Pascal
For i := 10 downto 0 do step 2
Begin
    Writeln(i);
End;
Java
for (int i = 10; i >= 0; i = i - 2)
{
    System.out.println(i);
}
