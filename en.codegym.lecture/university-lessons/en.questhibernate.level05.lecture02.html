Starting and stopping the service

Starting and stopping MySQL server is very easy. Since we installed it as a Windows Service, it will always be running for you.

If you need to stop it, then press Ctrl+Shit+Esc and select the Services tab:

By right-clicking on a service name , you can always start or stop it, as well as view its current running status.

MySQL Console

And although the MySQL server does not have a graphical interface, you can issue commands to it if you start the MySQL Console.

You need to go to the folder where MySQL was installed, in my case it is:

C:\Program Files\MySQL\MySQL Server 8.0\bin

And run the mysql.exe application with the following command:

mysql.exe -u root -p

The console will then prompt you to enter your root user's password. (mysql-root-password):

If the password is correct, you will see the following prompt:

And here you can execute various commands.

Let's write SELECT VERSION() ; and see the version of our SQL Server:

Important! Since commands are often multi-line, pressing Enter simply moves the cursor to the next line. To mark the end of a query, you need to put a semicolon at the end of it.
