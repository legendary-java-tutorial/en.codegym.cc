double type - real numbers

----------------------------------------

The double type To work with real (fractional) numbers, Java uses the double type. It takes up 8 bytes in memory (twice as many as the int type) and can store values ​​in the range -1.7*10308 to +1.7*10308. For comparison, the int type can store a value in the range -2*109 to +2*109. To the substance
