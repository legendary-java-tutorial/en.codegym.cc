Strategies for Avoiding DeadLock

----------------------------------------

Hello Amigo! I want to tell you about a couple of strategies for avoiding deadlocks. The best strategy is a well-thought-out architecture and a set of rules for when locks can be used (mutex acquisition) and in what order. The classic way to deal with the problem is to develop a lock hierarchy, establishing a rule that some locks can never be acquired in a state in which they are already acquired ...
